Nesse dialogo, o adminitrador pode selecionar açoes adicionais para um determinado usuário que pode ou não sobrescrever as ações já existentes do seu perfil.

A listagem da esquerda representam as ações disponíveis e que ainda não estão associadas ao usuário

O usuário pode atribuir uma ação clicando no botão ilustrado na imagem abaixo:

Ao mover uma ação, um dialogo é aberto pedindo para confirmar qual o seu tipo de acesso.

Existem dois tipos de acesso possíveis:

  • Bloquear: Significa que essa ação vai ficar bloqueada para esse usuário, ou seja, se um usuário já possui essa ação dentro do seu perfil, ela sera removida.
  • Permitir: Significa que essa ação vai ser atribuida a esse usuário, além das ações contidas no seu perfil.

A listagem da direita representa as ações já associadas ao usuário.
